Add unlimited custom pages to WordPress admin dashboard.

== Description ==

Try this convenient tool for adding custom pages to admin panel in the easiest way. You can add as many pages as you want and fill them with the necessary content using TinyMCE. 

http://www.youtube.com/watch?v=Ecvb7F2sATs

= Features =

* Add and manage unlimited custom pages
* Set capability level, page position, page icon
* Add content using TinyMCE
* Add page to existing pages

== Installation ==

1. Upload the `custom-admin-page` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings using your WordPress admin panel in "BWS Panel" > "Custom Admin Page".
